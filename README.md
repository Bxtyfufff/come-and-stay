开始之前公众号运营，我们以文章的方式共享给其他人，但由于微信账号的局限性，
每次大家发现新鲜事，写好文章后都要发给我，我再进行查看登入账号进行发布，
确实比较麻烦，而且还不能协同工作，所以决定把工作放到github上进行管理，
大家就可以自由的登入账号，写好文章，直接放到里面人，我在下面写个教程，
大家按照教程走一篇，后面就比较方方便了
